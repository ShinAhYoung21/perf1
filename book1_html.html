    strong
    굵은 글씨체
    u
    밑줄.underline.
    웹사이트에서 ctrl+u
    해당 페이지 소스코드 확인.
    p
    단락. paragraph.
    li
    구분 점이 있는 목차.list.
    ul
    구분 점이 없는 목차. unordered list.
    ol
    li과 같이 쓰면 목차에 숫자 넘버링 할 수 있다. ordered list.
    br
    줄바꿈
    img src="(주소)"
    이미지(image), 출처(source)
    title
    제목. 주소창에 표시되는 제목. 검색엔진은 이를 '책표지'처럼 인식한다.
    !doctype html
    관례적으로 써준다. '이 파일은 html로 작성되어 있다'라는 의미.
    head
    본문을 설명하는 부분을 감싸는 태그. 여닫을 수 있다.
    body
    본문. 소스코드의 주 내용을 작성하는 부분을 감싸는 태그. 여닫을 수 있다.
    a
    링크. anchoor(닻).
    target=""
    a 뒤에 쓰이는 속성(attribute)중 하나. 링크 문서를 새 창으로 열려면 _blank, 현재 페이지에 열려면 아무 값 입력 안 하면 된다(또는 _self입력).
    title=""
    이 역시 a 뒤에 쓰이는 속성 중 하나. 링크에 마우스 커서를 올리면 나타나는 '링크 제목'을 입력하는 태그.
    input type="checkbox"
    체크박스 삽입. (예시로만 봤던 태그라 한 번 써봤음ㅋㅋㅋ).
    blockquote
    들여쓰기 태그.
    embed
    퍼와서 삽입하는 태그. 활용해본 결과, 유튜브가 가장 퍼담기 좋다. 인스타는 굉장히 폐쇄적이라 embed와 쓰기엔 좋지 않다.
